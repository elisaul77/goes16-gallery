# Galería GOES-16 · Colombia

Galería web estática de imágenes satelitales GOES-16 sobre Colombia, generada automáticamente desde los servidores públicos del IDEAM / NOAA.

> Imágenes provistas por el [IDEAM](http://www.ideam.gov.co/) y el satélite [GOES-16 (NOAA)](https://www.goes.noaa.gov/). Uso exclusivamente informativo y educativo.
